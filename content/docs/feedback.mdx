# Feedback and Ratings

This section provides detailed information about the API endpoints for managing feedback and ratings.

## Create Facilitator Feedback

Creates feedback for a specific facilitator.

**Endpoint:** `POST /v1/facilitators/:id/feedback`

**Request Body:**

```json
{
  "session_enrollment_id": 1,
  "score": 5,
  "comment": "Excellent facilitator!"
}
```

**Example Request:**

```bash
curl -X POST -H "Content-Type: application/json" -d '{
  "session_enrollment_id": 1,
  "score": 5,
  "comment": "Excellent facilitator!"
}' "http://localhost:4000/v1/facilitators/1/feedback"
```

**Example Success Response (201 Created):**

```json
{
    "feedback": {
        "id": 1,
        "facilitator_id": 1,
        "session_enrollment_id": 1,
        "score": 5,
        "comment": "Excellent facilitator!",
        "created_at": "..."
    }
}
```

**Error Handling:**

*   If the facilitator does not exist, you will receive a `404 Not Found` response.
*   If the request body is invalid, you will receive a `422 Unprocessable Entity` response.

## List Facilitator Feedback

Retrieves all feedback for a specific facilitator.

**Endpoint:** `GET /v1/facilitators/:id/feedback`

**Example Request:**


**Example Success Response (200 OK):**

```json
            "created_at": "..."
```

**Error Handling:**

*   If the facilitator does not exist, you will receive a `404 Not Found` response.
